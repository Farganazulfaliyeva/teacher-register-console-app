# Teacher Register Console App

Teacher Register Console App is a simple Java console application for managing teacher records.

## Features

- Add a new teacher
- View all teachers
- Search teacher by ID or name
- Update teacher information
- Delete teacher record
- Simple console-based interaction
- Beginner-friendly project structure

## Technologies Used

- Java
- OOP principles
- Collections
- Console input/output

## Project Purpose

This project was built to practice Java fundamentals such as:

- Classes and objects
- Methods
- Constructors
- Lists
- Loops
- Conditional statements
- Basic CRUD operations

## How It Works

The user interacts with the application through the console menu.  
The system allows the user to manage teacher data by selecting different operations.

Example operations:

1. Add Teacher
2. Show All Teachers
3. Search Teacher
4. Update Teacher
5. Delete Teacher
6. Exit
